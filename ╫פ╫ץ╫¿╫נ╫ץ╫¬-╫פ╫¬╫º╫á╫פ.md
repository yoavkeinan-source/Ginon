# 🌿 גינון פרו — PWA

## הוראות העלאה לאינטרנט (חינמי)

### שלב 1: Firebase (סנכרון בין מכשירים + כניסה)

1. היכנס ל-https://firebase.google.com
2. לחץ "Get started" → צור פרויקט חדש (קרא לו "ginon-pro")
3. בתפריט שמאל: **Authentication** → Get started → Email/Password → Enable → Save
4. בתפריט שמאל: **Firestore Database** → Create database → Start in production mode → בחר אזור europe-west (קרוב לישראל)
5. לחץ על סמל ⚙️ (Project settings) → Web app → Register app
6. תראה קוד `firebaseConfig` — העתק אותו
7. פתח את הקובץ `index.html` ובחלק של `firebaseConfig` החלף את הערכים

### שלב 2: העלאה ל-Netlify (חינמי)

1. היכנס ל-https://netlify.com → Sign up (עם Google)
2. לחץ "Add new site" → "Deploy manually"
3. גרור את **תיקיית ginon-pwa** לתיבה
4. תקבל כתובת כמו: `amazing-ginon-12345.netlify.app`
5. ניתן לשנות שם: Site settings → Change site name

### שלב 3: כניסה מכל מכשיר

- פתח את הכתובת בכל דפדפן
- **iPhone**: Safari → שתף → "הוסף למסך הבית"
- **אנדרואיד**: Chrome → תפריט → "הוסף לדף הבית"
- **מחשב**: כפתור ➕ בשורת הכתובת של Chrome
